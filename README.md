# AUTOLIVE - Automation Platform for Viral Content

AUTOLIVE is a comprehensive automation platform that helps users discover viral videos, automatically generate clips, add subtitles and music, and upload videos to multiple social media platforms.

## Features

- 🔐 **User Authentication** - Secure login and registration with Google Sheets integration
- 📊 **Dashboard** - Overview of connected platforms and channel statistics
- 🔌 **Platform Integration** - Connect YouTube, TikTok, Instagram, and Facebook accounts
- ⚡ **Automation** - Configure video discovery and processing rules
- 📈 **Reports** - Track video performance across platforms
- 🐙 **GitHub Integration** - Backup automation scripts and files
- 🔄 **n8n Integration** - Trigger external automation workflows
- 📚 **User Guide** - Comprehensive documentation

## Technology Stack

- **Backend**: Node.js, Express.js
- **Frontend**: HTML5, CSS3, Vanilla JavaScript
- **Database**: Google Sheets API (for user data and reports)
- **APIs**: YouTube Data API, TikTok API, Instagram Graph API, Facebook Graph API
- **Integrations**: GitHub API, n8n, Railway
- **Utilities**: node-cron, winston, bcryptjs, jsonwebtoken
