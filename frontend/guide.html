Welcome to AUTOLIVE

AUTOLIVE is a powerful automation platform that helps you discover viral videos, automatically generate clips, add subtitles and music, and upload content to multiple social media platforms. This guide will walk you through all the features and how to use them effectively.

💡 Pro Tip: Start by connecting your social media accounts in the "Connect Platforms" section before setting up automation.

Getting Started

1. Create an Account

Visit the registration page and fill in your details. You'll need:

  • Full Name
  • Email address
  • Password (minimum 6 characters)
  • Phone number (optional)
  • Preferred language

2. Login to Your Dashboard

After registration, login with your email and password. You can also use Google Login for quick access.

3. Navigate the Dashboard

The dashboard shows an overview of your connected platforms and recent activity. Use the sidebar menu to access different features:

  • Dashboard - Overview and statistics
  • Connect Platforms - Link your social media accounts
  • Automation - Configure video discovery and posting rules
  • Reports - View performance analytics
  • GitHub Connection - Backup your automation scripts
  • n8n Integration - Connect external workflows
  • Guide - You are here!

Connecting Social Media Platforms

YouTube

  1. Go to the Connect Platforms page
  2. Enter your YouTube API Key
  3. Enter your YouTube Channel ID
  4. Click "Test Connection" to verify
  5. Click "Save & Connect" to store credentials

Where to find these:

  • API Key: Google Cloud Console → Enable YouTube Data API v3
  • Channel ID: Your YouTube channel URL (youtube.com/channel/CHANNEL_ID)

TikTok

  1. Go to the Connect Platforms page
  2. Enter your TikTok Access Token
  3. Click "Test Connection" to verify
  4. Click "Save & Connect" to store credentials

Instagram & Facebook

Follow similar steps using the Instagram Access Token and Facebook Page Token from their respective developer portals.

Configuring Automation

Video Discovery Settings

  1. Navigate to the Automation page
  2. Select a video category (Motivation, Technology, Sports, etc.)
  3. Set minimum views threshold (10K, 50K, 100K, etc.)
  4. Choose upload age filter (1h, 6h, 24h, 7d)

Video Processing Options

  • Clip Duration: 15, 30, or 60 seconds
  • Background Music: Automatically adds copyright-safe music
  • Subtitles: Auto-generated with single or dual language options
  • Titles & Hashtags: Automatically generated based on content

Platform Selection

Choose which platforms to upload to (YouTube, TikTok, Instagram, Facebook). Make sure you've connected these platforms first!

Scheduling

Set how many videos to upload per day and at what times. You can select multiple upload times for each day.

Starting Automation

Click the green "START AUTOMATION" button to begin. The system will:

  1. Discover viral videos based on your criteria
  2. Generate clips with the specified duration
  3. Add background music and subtitles
  4. Generate titles and hashtags
  5. Upload to selected platforms at scheduled times

Reading Reports

The Reports page provides comprehensive analytics for all your uploaded content.

Summary Cards

  • Total Videos: Number of videos uploaded
  • Total Views: Combined views across all platforms
  • Total Likes: Total engagement
  • Total Comments: Audience interaction
  • Subscriber Growth: New followers gained

Filtering Reports

Use the filters to narrow down data by:

  • Platform (YouTube, TikTok, Instagram, Facebook)
  • Time period (7 days, 30 days, 90 days, year, all time)

Exporting Data

Click the "Export CSV" button to download reports for offline analysis.

GitHub Integration

The GitHub Connection page allows you to backup your automation scripts and configurations.

Connecting GitHub

  1. Generate a Personal Access Token on GitHub (Settings → Developer settings → Personal access tokens)
  2. Enter your repository owner (username or organization)
  3. Enter your repository name
  4. Click "Connect Repository"

Features

  • Backup Files: Save your automation configurations
  • File Manager: Create and edit files directly in your repository
  • View History: See recent commits and backups

n8n Integration

The n8n Integration page lets you connect external automation workflows.

What is n8n?

n8n is an open-source workflow automation tool that can connect hundreds of services. AUTOLIVE can trigger n8n workflows when automation starts.

Configuration

  1. Enter your n8n webhook URL
  2. Optionally specify a workflow ID
  3. Add an API key if required
  4. Test the connection
  5. Save configuration

Testing Workflows

Use the workflow tester to send sample data and verify your n8n workflows are working correctly.

Troubleshooting

Common Issues

🔌 Platform Connection Failed

Verify your API keys and tokens are correct and have the necessary permissions. Some platforms require specific scopes to be enabled.

⚡ Automation Not Starting

Ensure you have at least one platform connected and have configured all required settings. Check that your scheduling times are valid.

📊 No Reports Showing

Reports only appear after videos have been uploaded. Give it some time after your first automation run.

🐙 GitHub Connection Issues

Make sure your Personal Access Token has the correct permissions (repo scope). Verify repository name and owner are correct.
